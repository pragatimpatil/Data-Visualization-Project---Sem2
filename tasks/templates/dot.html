DOT PLOT

Introduction

Dotplots are a popular visualization method in bioinformatics for comparing two sequences and locating commonality. Plotting one sequence down the x-axis and the other along the y-axis, then placing a dot at each location where the two sequences coincide, is the core notion behind this technique. The resulting plot gives you a simple, quick approach to spot places where the sequences are similar.


Generating a Dot plot Matrix

To create a Dot plot matrix, two sequences are compared by scanning them across a grid and marking the locations where they match. The outcome is a matrix, where each cell denotes a location in one sequence and a different place in a different sequence.

The cell is indicated with a dot or another symbol if the two places match.

The scatter() function from Matplotlib is used by the function to produce a new figure and plot the dots on the scatter plot. The indices of the non-zero elements in the dot plot matrix define the placements of the dots.
The set_xlabel(), set_ylabel(), and set_title() functions are used to set the x- and y-axis labels as well as the title. The set_xticks(), set_yticks(), set_xticklabels(), and set_yticklabels() routines are used to set the tick positions and labels. The plot is then shown on the screen and saved to the designated file using the savefig() and show() functions, respectively.
Dot plot matrices produced by the dot plot algorithm can be visualized using this function for a variety of tasks including sequence alignment, genome comparison, and motif discovery. Dot plot visualization is made versatile and programmable with the usage of Matplotlib.

seqA="peter piper picked a peck of pickled peppers"
seqB="a peck of pickled peppers peter piper picked"
dp = dotplot(seqA,seqB,5,4)
dotplot2Graphics(dp, 'Sequence A', 'Sequence B', "Peter Piper’s first dotplot", "mydotplot.png")

This code generates a dot plot graphic using the dotplot2Graphics function in Python's Matplotlib library. The dotplot2Graphics function takes as input a dot plot matrix dp and labels for the two sequences being compared, labelA and labelB, as well as a title for the plot and the name of the file to save the plot as. In this particular example, the dot plot matrix dp is generated from two DNA sequences, seqA and seqB, using the dotplot function. The function then calls dotplot2Graphics to create a dot plot visualization of the matrix. The x-axis represents the positions in sequence B, while the y-axis represents positions in sequence A. The dots represent matches between the two sequences.
